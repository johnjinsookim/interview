# Azure AI Foundry Interview MVP

This repository contains interview materials for the Azure AI Foundry team.

## Goal
Design and implement an MVP product that enables users to get started with out-of-the-box observability when creating a new agent.

## Contents
- `docs/interview/`: product strategy, requirements, and implementation notes
- `prototype/interview/`: interactive prototype assets and pages
- `skills/`: supporting skill write-ups used in the process
